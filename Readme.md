# How to start the project?

<p>Clone the repository with: git clone git@github.com:henrique1758/tasker-server.git</p>

<p>Install the dependencies with the command:</p>

```sh
yarn
```
<p>To execute the server run the command:</p>

```sh
yarn dev
```
